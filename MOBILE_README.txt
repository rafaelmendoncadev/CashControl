📱 CASHCONTROL NO CELULAR - INSTRUÇÕES RÁPIDAS
============================================

🚀 OPÇÃO MAIS FÁCIL: KIVY LAUNCHER
----------------------------------

1. 📱 Instale "Kivy Launcher" da Play Store
   https://play.google.com/store/apps/details?id=org.kivy.pygame

2. 💻 No PC, execute:
   python preparar_kivy_launcher.py

3. 📂 Copie a pasta criada para seu celular em:
   /sdcard/kivy/cashcontrol/

4. 🚀 Abra Kivy Launcher no celular e toque em "cashcontrol"

✅ PRONTO! Seu app está funcionando!

🔧 OPÇÃO AVANÇADA: APK PROFISSIONAL
-----------------------------------

1. 🐧 Instale WSL no Windows ou use Linux

2. 📦 Execute no terminal:
   chmod +x compilar_android.sh
   ./compilar_android.sh

3. 📱 Instale o APK gerado no celular

⏱️ Primeira compilação demora 30-60 minutos

📋 ARQUIVOS IMPORTANTES:
- guia_completo_mobile.py    - Guia interativo completo
- preparar_kivy_launcher.py  - Prepara arquivos para teste rápido  
- compilar_android.sh        - Compila APK profissional
- buildozer.spec            - Configuração do APK

🤔 QUAL ESCOLHER?
-----------------
• Teste rápido → Kivy Launcher
• App para distribuir → Buildozer (APK)

💡 DICA: Comece com Kivy Launcher para testar!

🆘 AJUDA:
Execute: python guia_completo_mobile.py
